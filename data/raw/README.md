# Raw Data

This folder contains the original dataset used for the BlueLine Logistics Operational Analysis project.

The raw CSV files have been compressed into **`raw_data.zip`** to reduce repository size and simplify file management.

### Dataset Source
**Synthetic Logistics Operations Database (2022–2024)**  
Author: Yogape Rodriguez (2025)  
Source: Kaggle

https://www.kaggle.com/datasets/yogape/logistics-operations-database

### Contents

The compressed archive contains the following 14 original tables:

- Customers
- Delivery Events
- Driver Monthly Metrics
- Drivers
- Facilities
- Fuel Purchases
- Loads
- Maintenance Records
- Routes
- Safety Incidents
- Trailers
- Trips
- Truck Utilization Metrics
- Trucks

> **Note:** These files are the original, unmodified datasets. All data cleaning, validation, and analysis were performed on SQL staging tables. The cleaned datasets used for analysis are available in the `processed data` folder.
